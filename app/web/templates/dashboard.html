Jira Personal Access Token

How to create a Jira PAT
  1. Open Jira in your browser.
  2. Click your avatar/profile icon at the top right.
  3. Open Profile.
  4. Select Personal access tokens in the left menu.
  5. Click Create token.
  6. Enter a token name, for example QA AI Platform.
  7. Optionally set an expiration date.
  8. Click Create and copy the token.

The token will be stored in this browser local storage only. Do not use this option on a shared machine.
